# Instrucciones para las Imágenes del Cuerpo

Para que el componente BodyMap funcione correctamente, necesitas colocar las siguientes imágenes en esta carpeta (`public/images/`):

## Imágenes Requeridas

1. **body-frontal.png** (o .jpg)
   - Vista frontal del cuerpo humano
   - Sistema muscular visible
   - Fondo blanco
   - Formato: PNG o JPG

2. **body-posterior.png** (o .jpg)
   - Vista posterior (de atrás) del cuerpo humano
   - Sistema muscular visible
   - Fondo blanco
   - Formato: PNG o JPG

3. **body-lateral.png** (o .jpg)
   - Vista lateral (de lado) del cuerpo humano
   - Sistema muscular visible
   - Fondo blanco
   - Formato: PNG o JPG

## Ubicación de las Imágenes

Las imágenes deben estar en:
```
phonealivia/public/images/
├── body-frontal.png
├── body-posterior.png
└── body-lateral.png
```

## Recomendaciones

- **Tamaño**: Las imágenes deben tener una proporción similar (altura mayor que ancho)
- **Resolución**: Mínimo 800x1200px para buena calidad
- **Formato**: PNG con transparencia o JPG con fondo blanco
- **Orientación**: Cuerpo completo, de pie, con brazos ligeramente abiertos

## Nota

Si las imágenes no están disponibles, el componente mostrará un mensaje indicando que deben colocarse en esta ubicación.
