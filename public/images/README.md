# Imágenes del Cuerpo Humano

Coloca aquí las imágenes del cuerpo humano:
- `body-frontal.png` o `.jpg` - Vista frontal
- `body-posterior.png` o `.jpg` - Vista posterior (de atrás)
- `body-lateral.png` o `.jpg` - Vista lateral (de lado)

Las imágenes deben mostrar el sistema muscular del cuerpo humano con fondo blanco.
